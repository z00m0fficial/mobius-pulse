# Pulse Metrics

Pulse owns Mobius organizational intelligence metrics.

## Core Metrics

- OIES: organizational intelligence efficiency.
- MIS: Mobius intelligence score.
- VCI: voice confidence index.
- Decision Latency: time from signal intake to routed recommendation.
- Memory Reuse Rate: share of new work using prior verified memory.
- Department Health: availability, quality, performance, and verification rate.
- Learning Velocity: rate of verified knowledge growth.

## Rule

No metric should appear without source, confidence, and explanation.

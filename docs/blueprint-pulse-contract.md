# Blueprint Pulse Contract

## Role
Mobius Pulse supplies Organization Pulse and Learning Velocity signals for Dynamic Organization Blueprints.

## Blueprint Inputs
- verifiedMemoryCount
- activeDepartmentCount
- riskSignals
- recommendationCount
- dnaState

## RC-1 Deterministic Metrics
- Organization Pulse: Healthy, Learning, or Needs Attention
- Learning Velocity: percentage derived from verified memories and active departments

## Connected Repositories
- mobius-command-center
- mobius-memory
- mobius-atlas-dna
- mobius-dynamic-organization-builder

# Mobius Pulse

Organizational intelligence metrics, analytics, scoring, and executive insights for the Mobius Organizational Intelligence Platform.

## Mission

Measure how well an organization learns, remembers, routes, verifies, and improves.

## Core Metrics

- OIES: Organizational Intelligence Efficiency Score
- MIS: Mobius Intelligence Score
- VCI: Voice Confidence Index
- Decision Latency
- Memory Reuse Rate
- Department Health
- Repository Health Trends
- Organizational Learning Velocity
- Workflow Efficiency

## Boundary

Pulse measures. Atlas reasons. Memory remembers. Router orchestrates. Sonic listens. Command Center displays.
